# Traffic Light Controller using 8051

This project is a Traffic Light Controller designed using the 8051 microcontroller and simulated in Proteus using Assembly language.

## Project Overview

The system controls traffic lights in a sequential manner similar to real-world traffic signal systems. LEDs are used to represent Red, Yellow, and Green traffic lights, and the timing sequence is controlled through the 8051 microcontroller.

## Features

- Traffic light sequencing
- 8051 microcontroller based design
- Proteus simulation
- Assembly language programming
- Simple embedded systems project

## Components Used

- 8051 Microcontroller
- LEDs
- Resistors
- Crystal Oscillator
- Capacitors
- Power Supply
- Proteus Software

## Software Used

- Proteus 8
- 8051 Assembly Language

## Files Included

- Proteus project files
- Assembly code (`main.asm`)
- Simulation screenshots
- Working demonstration video

## Working

The microcontroller switches the traffic lights automatically with predefined delays:

1. Red Light ON
2. Yellow Light ON
3. Green Light ON
4. Sequence repeats continuously

## Output

The Proteus simulation demonstrates proper traffic signal operation using LEDs.
